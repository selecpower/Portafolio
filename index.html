Hola, soy Francisco

Desarrollador Fullstack

Con formación como Ingeniero en Software y pasión por el desarrollo backend, me dedico a crear soluciones robustas y escalables que impulsan la eficiencia de procesos reales. Disfruto diseñar arquitecturas limpias que conectan de forma fluida con bases de datos y servicios externos.

Me especializo en construir APIs RESTful y sistemas de gestión usando Java con Spring Boot y PHP con Laravel. Cuando el proyecto lo requiere, complemento mi trabajo con interfaces en React, asegurando una experiencia de usuario moderna y coherente.

Me considero autodidacta, orientado al detalle y comprometido con la entrega de valor. Disfruto optimizar bases de datos relacionales, mantener un flujo de trabajo limpio en Git y colaborar en equipos ágiles bajo metodologías como Scrum.

Portafolio

JavaWorks - Ecommerce

Plataforma ecommerce con tematica de dulces tradicionales y exoticos, construida con Java y Spring Boot para administrar productos, categorías, pedidos y usuarios. Incluye un sistema de autenticación, panel administrativo para CRUD de productos, carrito de compras, control de stock y procesamiento de pedidos. La arquitectura está basada en MVC con un enfoque RESTful, integrando MySQL para persistencia de datos y Bootstrap en el frontend para una experiencia visual moderna y accesible.

  • Java
  • Spring Boot

Estado: Funcional

Sistema de Gestión de Almacén

Sistema web desarrollado en Laravel que permite gestionar eficientemente un almacén de herramientas y materiales dentro de una empresa de mantenimiento. La plataforma incluye autenticación con distintos roles (administrador y empleado), control de inventario, registro y edición de herramientas, así como un módulo para gestionar peticiones internas. Se incorporó Bootstrap para una interfaz clara y ordenada, orientada al uso en escritorio, además de un modelo de base de datos relacional que garantiza integridad y escalabilidad.

  • Laravel
  • PHP
  • MySQL
  • Bootstrap

Estado: Funcional (pendiente: stock dinámico y responsive)

Gestor de Productos Fullstack

Aplicación fullstack desarrollada con React en el frontend y Spring Boot en el backend para la gestión integral de productos, clientes, carrito y pedidos. Permite realizar operaciones CRUD desde un panel administrativo, añadir productos al carrito desde el catálogo, y generar tickets de las compras. El backend expone servicios REST y utiliza H2 como base de datos embebida para pruebas, mientras que el frontend implementa rutas protegidas, hooks personalizados y consumo dinámico de la API.

  • React
  • Spring Boot
  • H2 Database
  • Postman
  • Git

Estado: En desarrollo
